实现功能 <br/>
	1、可以改变记录的排列布局；<br/>
	2、可以添加，收藏，分享，删除记录<br/>
	3、可以改变记录字体的大小；<br/>
	4、本地数据的合理保存，避免信息以及配置的丢失；<br/>
